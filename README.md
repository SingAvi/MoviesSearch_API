# MoviesSearch_API

## API KEY
  Go to [OMDb](http://www.omdbapi.com/apikey.aspx) and fetch your API_KEY .
  
### Required Library
 - Json
 - urllib2 / urllib3
 
### Installing Libraries

- Json

> For Conda


conda install -c jmcmurray json


> For simple Pip


pip install json


- Urllib2/3

> For Conda


conda install -c ulmo urllib3


> For simple Pip


pip install urlib2/3
